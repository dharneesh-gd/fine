HOW TO TEST:
-----------

1. DESKTOP BROWSER:
   - Open any page in Chrome/Firefox/Safari
   - Press F12 to open Developer Tools
   - Click device toolbar icon (or Ctrl+Shift+M)
   - Select different device presets (iPhone, iPad, etc.)
   - Test different screen sizes

2. MOBILE DEVICE:
   - Open the website on your phone/tablet
   - Tap the hamburger menu (☰) in header
   - Verify menu opens and closes properly
   - Test all buttons are easy to tap
   - Check text is readable without zooming
   - Try both portrait and landscape modes

3. SPECIFIC TEST PAGE:
   - Open mobile-test.html
   - Shows current device info
   - Lists all responsive features
   - Displays active breakpoint
   - Resize window to see real-time updates
